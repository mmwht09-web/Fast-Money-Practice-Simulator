FAST MONEY PRACTICE SIMULATOR
=============================

DESCRIPTION:
This simulator allows you to practice fast-money decision-making in a safe, educational environment. 
All results are simulated; no real money is used.

FILES INCLUDED:
1. fast_money_simulator.py – Python version
2. fast_money_simulator_web.html – Web version (runs in any browser)

HOW TO RUN:

Python Version:
1. Make sure Python 3 is installed on your computer.
2. Open terminal or command prompt.
3. Navigate to the folder containing fast_money_simulator.py
4. Run: python fast_money_simulator.py
5. Follow on-screen instructions.

Web Version:
1. Open fast_money_simulator_web.html in any modern web browser.
2. Check the disclaimer box.
3. Click "Run Session" to simulate trades.
4. Optional: Click "Download CSV" to save results.

DISCLAIMER:
This product is for educational and simulation purposes only.
It does not provide financial, investment, or trading advice.
No real market data is used.
Results do not represent real-world outcomes.
Use at your own risk.
